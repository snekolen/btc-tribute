    Muse

    Muse is a discord bot that sends music based on user-inputted countries and decades and keeps track of top search terms and recently liked songs in a server. It is coded in Python and uses various APIs and classes to do the aforementioned tasks. The links below include a repository and server invite link.

    Repository: https://github.com/snekolen/muse

    MazeBot

    MazeBot is maze traversal robot that uses LIDAR sensors to navigate through a maze. It is a collaborative project between students of various majors whose skills lie within the group's computer science, electrical, and mechanical engineering subgroups. This project is currently a work in progress.
